Репозиторий с проектами и решенными задачами:

1) Папка "K-C final project" содержит финальный проект курса "Аналитик данных" Karpov/Courses.
1) Папка "Kinopoisk. Data-preparation&analysis" содержит проект парсера данных с сайта Кинопоиск. Реализованы сбор, хранение, обработка и анализ данных.
2) Папка "MySQL" содержит проект создания БД Instagram c помощью MySQL 8.0 + несколько SQL-запросов к созданной БД. Дамп получивщейся БД приложен.
3) Папка "Python" содержит решенные задачи по работе с данными. Темы: агрегация, группировка, визуализация данных, сводные таблицы, работа с API, ClickHouse + Python
4) Папка "Tableau" содержит небольшую задачу на построение визуализации данных в Tableau + выводы на основании визуализации.

В каждой папке имеется текстовый файл с более детальным описанием содержимого. К проектам приложены пояснительные записки в формате .pdf
